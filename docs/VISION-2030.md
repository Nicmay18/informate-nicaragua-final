# VISION 2030 — Nicaragua Informate

## Propósito

Nicaragua Informate utilizará inteligencia artificial para ayudar a periodistas a producir noticias más rápidas, más completas y mejor documentadas.

> La IA nunca sustituye el criterio editorial.  
> La IA recomienda. El editor decide.

## Prioridades

### 2026

- Estabilizar Editor IA

### 2027

- Crecer audiencia
- Mejorar productividad
- Optimizar Discover

### 2028

- Automatización editorial

### 2029

- Asistente de investigación

### 2030

- Plataforma editorial inteligente

## Regla para todos los proyectos futuros

### Ciclo de desarrollo

1. Idea
2. Documento (roadmap)
3. Validación
4. Desarrollo
5. Producción
6. Observación
7. Métricas
8. Mejoras

> Nunca volver directamente de **Producción → Desarrollo** porque apareció un caso aislado.

## Cambio de enfoque

Al inicio de este proyecto, el objetivo era:

> Construir un Editor IA.

Hoy el objetivo es otro:

> Construir el mejor medio digital de Nicaragua utilizando el Editor IA como herramienta.

Ese cambio es el que empieza a traducir el trabajo en resultados reales: más publicaciones de calidad, más audiencia y, con el tiempo, mejores ingresos. El software deja de ser el fin y pasa a ser el medio para hacer crecer Nicaragua Informate.

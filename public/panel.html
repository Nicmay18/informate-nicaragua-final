Redacción con IA

Genera noticias profesionales estilo agencia internacional.

Asistente Editorial Gemini

Configura los parámetros y genera una noticia con estructura de pirámide invertida, tono Reuters/AP y optimización AdSense.

Prompt Sistema (Avanzado)

Actúa como un corresponsal senior de una agencia internacional de noticias (tipo Reuters o AP) destacado en Nicaragua. Redacta una noticia basada en el título: '${titulo}'. DIRECTRICES DE REDACCIÓN: 1. Lead de Impacto: Comienza con un párrafo que responda las 5W, situando la acción en el departamento de ${departamento}. 2. Contexto Nacional: Explica brevemente cómo este hecho afecta el panorama social o económico de Nicaragua en el contexto actual. 3. Neutralidad Absoluta: Mantén un tono formal, evitando adjetivos sensacionalistas para cumplir con las políticas de Google AdSense. 4. Estructura de Pirámide Invertida: Organiza la información de lo más importante a lo menos relevante. 5. Vocabulario: Usa un español neutro pero reconoce términos geográficos y culturales de Nicaragua. 6. Cierre Profesional: Termina con datos de contexto o antecedentes del caso. 7. Extensión: Genera entre 400 y 600 palabras para cumplir con los requisitos de calidad de AdSense. RESTRICCIÓN: Entrega solo el cuerpo de la noticia. No incluyas 'Aquí tienes la noticia' ni otros comentarios.

Este prompt se envía automáticamente a Gemini con los parámetros que selecciones arriba.

    Guías Editoriales

    Protocolos de publicación, SEO y monetización.

    Rank Math SEO (Recomendado sobre Yoast)

    Rank Math es superior para sitios de noticias porque permite configurar el Schema Markup de NewsArticle, indicándole a Google que tu contenido es una noticia real y mejorando tu posicionamiento en Google News.

    • Configura el Schema Type: NewsArticle en cada publicación.
    • Activa "Index Now" para que Google indexe tus noticias en minutos.
    • Usa el SEO Score de Rank Math: apunta a 80+ puntos por artículo.
    • Configura el Sitemap de noticias (news-sitemap.xml) para Google News.

    Editor Gutenberg (Bloques Nativos)

    No uses Elementor ni constructores pesados para noticias. Gutenberg es más rápido, liviano y Google AdSense prefiere sitios con buen Core Web Vitals.

    • Usa el bloque "Párrafo" nativo. No copies desde Word.
    • Inserta bloques "HTML personalizado" solo si es necesario.
    • Activa "Carga diferida" (lazy loading) en imágenes nativamente.

    Nano Banana 2 (Gemini Image Generation)

    Para generar imágenes de portada cuando no cuentas con fotos propias:

    Prompt recomendado: "Fotografía de estilo fotoperiodismo de [escena específica en Nicaragua], luz natural de atardecer, alta resolución, aspecto realista, sin texto ni marcas de agua, composición periodística"
    • Siempre especifica "sin texto" para evitar problemas de accesibilidad.
    • Usa la relación de aspecto 16:9 para imágenes destacadas.
    • Añade siempre pie de foto (caption) con crédito: "Foto: Redacción NI".

    Instant Images (Plugin WordPress)

    Plugin para buscar fotos de stock gratuitas directamente desde el panel de WordPress sin salir del editor.

    • Fuentes integradas: Unsplash, Pexels, Pixabay.
    • Filtra por orientación: Horizontal para noticias.
    • Descarga directa al media library con atributo ALT automático.

    Requisitos para Aprobación AdSense en Nicaragua

    • Extensión mínima: Cada noticia debe tener entre 350 y 500 palabras como mínimo. Google considera contenido más corto como "poco valioso" o "thin content".
    • Páginas legales obligatorias: Debes tener visibles las páginas de "Política de Privacidad" y "Quiénes Somos" (enfocada en tu equipo editorial en Nicaragua).
    • Contenido prohibido: No uses IA para redactar sobre temas extremadamente violentos, gore, contenido que incite al odio o desinformación. AdSense banea cuentas permanentemente por esto.
    • Originalidad: El contenido generado por IA debe ser revisado, fact-checking aplicado y enriquecido con datos locales reales.
    • Navegación clara: Menús funcionales, categorías definidas y sin páginas rotas.

    Estructura de un Artículo "Internacional"

    Para que tu diseño se vea como un medio de comunicación grande y profesional, sigue este orden obligatorio:

    1. Imagen destacada: Foto de alta calidad con pie de foto. Ej: "Catedral de León, Nicaragua. Foto: Redacción".
    2. Dateline: (MANAGUA / NICARAGUA) en negrita al inicio del texto, justo después del título H1.
    3. Lead: Primer párrafo de 25-35 palabras que responda las 5W.
    4. Cuerpo: Párrafos cortos de no más de 4 líneas. Espaciado generoso.
    5. Bloque de relacionados: Inserta "Noticias relacionadas" a mitad del texto para retener al lector y reducir tasa de rebote.
    6. Cierre: Datos de contexto, antecedentes o próximos pasos. Nunca termines abruptamente.

    WP All Import

    Si tienes noticias en Excel o recibes feeds RSS de agencias:

    • Importa masivamente desde CSV/XML manteniendo el formato de campos personalizados.
    • Mapea: título → post_title, contenido → post_content, imagen → _thumbnail_id.
    • Programa importaciones recurrentes para agregadores de noticias.

    Make.com (Integromat) — Flujo Pro

    Automatización avanzada sin código:

    • Trigger: Nueva fila en Google Sheets (título + categoría).
    • Action 1: Gemini API genera el cuerpo de la noticia.
    • Action 2: WordPress REST API crea el post como borrador.
    • Action 3 (opcional): Notificación por Telegram al editor jefe para revisión.

    Beneficio: Reduce el tiempo de producción de 45 minutos a 5 minutos por noticia.
